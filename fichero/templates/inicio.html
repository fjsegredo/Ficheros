Bienvenido a Ficheros, una base de datos para hacer fichas bibliográficas.

Para contribuir debes:

1. Crear un usuario

2. Loguearte con tu nombre de usuario y contraseña

Una vez hecho eso puedes:

1. Agregar libros, capítulos o artículos a la base de datos.

2. Ver los libros que existen en nuestra base.

3. Fichar cualquier libro, capítulo o artículo. Cada ficha debe tener un texto, y puedes adjuntarle una imagen. Las posibilidades son infinitas.

4. Ver todas las fichas hechas a un libro/capítulo/artículo particular.

5. Hacer búsquedas de libros, capítulos y artículos filtradas según ciertos parámetros

6. Hacer búsquedas de fichas según el contenido.

7. Editar tu usuario, agregar información personal a tu perfil, y cambiar tu contraseña.

8. Buscar otros usuarios.
